# 🎬 Movie Success Prediction (Basic ML Project)

## 📌 Overview

This project predicts whether a movie will be successful based on limited features such as **Metascore and Genre**. It demonstrates the application of basic data science techniques including data cleaning, exploratory data analysis, and simple machine learning.

---

## 🛠️ Tech Stack

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn

---

## 📂 Dataset

* Movie dataset containing:

  * Metascore
  * Genre
  * Success label (hit/flop or revenue-based classification)

---

## 🔍 Key Steps Performed

### 1️⃣ Data Cleaning

* Handled missing values
* Processed categorical data (Genre encoding)

---

### 2️⃣ Exploratory Data Analysis (EDA)

* Analyzed relationship between:

  * Metascore and success
  * Genre and success
* Visualized trends using plots

---

### 3️⃣ Feature Selection

* Used limited features:

  * Metascore
  * Genre

---

### 4️⃣ Model Building

* Built a **basic classification model** using simple machine learning techniques

---

### 5️⃣ Model Evaluation

* Evaluated using:

  * Accuracy
  * Basic performance metrics

---

## 📊 Results

* Model successfully predicted movie success using limited features
* Showed how even basic features can provide insights into outcomes
* Highlighted limitations due to restricted feature set

---

## 🚀 How to Run

1. Clone the repository
2. Install dependencies from `requirements.txt`
3. Open the notebook in Jupyter/Colab
4. Run all cells

---

## 💡 Key Learnings

* Feature selection plays a crucial role in prediction accuracy
* Limited features restrict model performance
* Basic machine learning models are useful for understanding patterns

---

## 📌 Limitations

* Model uses only **Metascore and Genre**, limiting accuracy
* Does not include important factors like budget, cast, or marketing

---

## 📌 Future Improvements

* Include more features (budget, revenue, cast)
* Use advanced models
* Improve prediction accuracy

## 📎 Conclusion

This project demonstrates a foundational approach to predicting outcomes using basic data science techniques and highlights the importance of feature selection in machine learning.
